mcu-uuid-log
============

Description
-----------

Microcontroller logging framework library

Purpose
-------

Provides a framework for handling log messages. Thread-safe on the ESP32
but cannot be used from an interrupt context.

Dependencies
------------

* `mcu-uuid-common <https://mcu-uuid-common.readthedocs.io/>`_

Refer to the :download:`library.json <../library.json>` file for more
details.
